SpecRunner

A simple Windows GUI for running nspecs

View on GitHub Download .zip Download .tar.gz

If you are into unit testing and have tried the NSpec then for sure you missed the GUI similar to Nunit for running the unit tests. So this is it fellas. SpecRunner is a small windows application to run your specs.

Salient Features

  • Multiple Assembly support
  • Saving/opening project files
  • Debug the tests by attaching the process to Visual Studio
  • Full Stack Trace for each failed test
